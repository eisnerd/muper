Muper

Music player client for windows, for the MPD music playing server

Download this project as a .zip file Download this project as a tar.gz file

Forked from WpfMpdClient, this is a client for the MPD music playing server, developed using C# and WPF.

  • Browse the database
  • Browse saved playlists
  • View lyrics
  • Scrobble last.fm
  • Download album art from last.fm
  • Watch popups on track change by the tray icon
  • Use multimedia keys (play, pause, next track, previous track)
keyboard-centric bug tracker continuous integration server
